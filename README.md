# Sistemas-Informacao
🏗️ Arquitetura Empresarial — Macroanálise Team-it
Projeto de Sistemas de Informação | UTAD 2025/2026

Este repositório contém a macroanálise do Sistema de Informação da Team-it, uma consultora tecnológica integrada no ecossistema moOngy. O objetivo central foi definir e propor uma arquitetura empresarial que garanta o alinhamento estratégico entre a tecnologia e os objetivos do negócio.

🤝 Contacto Profissional e Experiência
Um dos pilares deste trabalho foi a componente prática e o envolvimento com o meio empresarial:

Contacto com a empresa Reuniões: Estabelecemos uma ligação próxima para compreender as dinâmicas reais de mercado e os desafios de gestão de informação.

Presença com o CEO: Tivemos a oportunidade de estar presentes em reuniões estratégicas com o CEO, permitindo obter uma visão executiva sobre a liderança e os macroprocessos da organização.

🛠️ Estrutura do Projeto
A análise foi decomposta em quatro camadas fundamentais de arquitetura empresarial:

1. 💼 Arquitetura de Negócio
Modelação detalhada dos processos nucleares da empresa, incluindo Recrutamento (IT Talent), Gestão Comercial e Acompanhamento de Consultores, utilizando a notação BPMN e a ferramenta SAP Signavio.

2. 📊 Arquitetura de Informação
Identificação de entidades de dados críticas (Clientes, Candidatos, Ocorrências) e definição da sua interação com os processos através de uma Matriz CRUD (Create, Read, Update, Delete).

3. 🖥️ Arquitetura de Aplicações
Mapeamento do ecossistema aplicacional assente em soluções SaaS, com destaque para o Microsoft 365, CRM Vitae e ERP Primavera/Cegid.

4. 🌐 Arquitetura Tecnológica
Análise da infraestrutura Cloud-First via Microsoft Azure, garantindo a segurança, redundância global e escalabilidade dos sistemas.
